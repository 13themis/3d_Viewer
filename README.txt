3DViewer

1. Установка

Для установки необходимо перейти в папку src и в терминале написать make install. После, программа соберется и автоматически запустится.

2. Удаление
Для удаления, необходимо перейти в папку src и в терминале написать make unistall. После чего, все сторонние файлы программы удалятся.

3. Архивация
Упаковать программу можно в тар архив можно командой make dist.

4. Возможности программы.
- Приложение поддерживает сохранение и загрузку сохраненных файлов, в которую входит сохранение цвета фона, цвета модели, цвета вершин и их типов, расположения и ротации, и размера модели.
- Программа работает с .obj файлами где вершины и линии не превышают больше 3х позиций.
- Программа может записывать действия пользователя, конвертируя в .gif. Так же есть возможность делать скриншоты.
- Есть отображение пунктирных линий, отображение текстуры, вершин(ничего/ круги/ квадраты), изменение цвета модели, вершин и заднего фона.
- Изменение ротации модели, ее размеров, и перемещение. Реализована возможность осмотра модели при помощи wasd и контроля мыши.
- Реализованны 2 типа камер для отображения модели.
- Есть возможность сохранения текущего состояния программы.
